Note: Internet Explorer 9 only supports .eot fonts.
With CSS3, websites can finally use fonts other than the pre selected "web-safe" fonts.
There is CSS code to get this font for document: @font-face { font-family: myFont; src: url('Files/Sansation_Light.ttf'); src: url('Files/Sansation_Light.eot'); /* IE9 */ }
