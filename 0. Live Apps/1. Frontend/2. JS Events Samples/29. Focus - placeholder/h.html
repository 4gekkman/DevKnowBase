Плейсхолдер для input типа text, работающий на основе событий focus/blur
  • В текстовых полях ниже ты видишь плейсхолдеры
  • Эти плейсхолдеры добавлены JS-скриптом всем эл-там input типа text, содержащим класс 'placeholder'.
  • Текст для плейсхолдера берется из польз. аттр. data-placeholder элемента input
  • Нажми на текстовое поле, и он исчезнет
  • Нажми где-нибудь в другом месте, тестовое поле потеряет фокус, и плейсхолдер снова появится
