Фильтрация ввода
  • Это поле для ввода возраста.
  • Вводить в него можно только цифры.
  • Если результирующая цифра меньше 0 или больше 150, она автоматически изменяется на ближайшую допустимую.
  • Учтена возможность того, что данные могут быть вставлены в input не с помощью клавиатуры, а другими способами.
