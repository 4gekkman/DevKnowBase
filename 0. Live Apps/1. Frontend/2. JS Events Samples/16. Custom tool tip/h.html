Всплывающие подсказки
  • Понаводи курсор на все 4 представленных блока, и наблюдай всплывающие подсказки.
  • Подсказка должна всплывать в 10px над элементом точно посередине
  • Размер всплывющей подсказки рассчитывается автоматически по содержимому HTML
  • Обрати внимание, что всплывает подсказка только для самого вложенного элемента
  • Обрати внимание, что при наведении на YELLOW подсказка не помещается посередине над элементом по оси X (иначе бы она вылезла за край экрана), и поэтому автоматически смещена вправо.
  • Обрати внимание, что при наведении на GREEN, подсказка не помещается над элементом по оси Y, иначе бы она вылезла за край экрана. Поэтому она показывается снизу на расстоянии 10px. Также, она не помещается по оси X ровно посередине элемента, поэтому она автоматически смещена влево.
