Шаг 1

Вводите значения, которые будут отправлены на сервер с XHR-запросом:

Name
Age
File

Шаг 2

Здесь в режиме реального времени формируется строка-тело в формате multipart/form-data:

""

Шаг 3

Сформированную на этапе "Шаг 2" строку теперь можно отправить на сервер, для этого нажми на кнопку ниже:
  • Запрос будет сделан методом POST (строку в формату multipart/form-data нельзя послать методом GET, а можно только методом POST).
  • Сервер в автоматическом режиме (используя массивы $_POST и $_FILES) распарсит полученные данные. А затем пришлёт ответ, который отобразится под кнопкой.
