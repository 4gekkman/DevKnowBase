Шаг 1

Вводите значения, которые будут отправлены на сервер с XHR-запросом:

Name
Surname
Age

Шаг 2

Здесь в режиме реального времени формируется строка-тело в формате JSON:

""

Шаг 3

Сформированную на этапе "Шаг 2" строку теперь можно отправить на сервер, для этого нажми на кнопку ниже:
  • Запрос будет сделан методом POST (строку в формате JSON в 99.9% случаев посылают методом POST, поскольку если отправлять методом GET в query string, то JSON-строка подвергнется еще и автоматическому URL-кодированию, что нам не нужно).
  • Сервер получит необработанные POST-данные, распарсит их с помощью функции json_decode, получив массив, и вернёт его клиенту. Массив отобразится ниже, под кнопкой.
