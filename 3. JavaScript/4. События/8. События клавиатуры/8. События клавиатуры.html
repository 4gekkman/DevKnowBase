Перевод символа в верхний регистр
  • Вводи буквы в нижнем регистре в поле ниже
  • Они будут автоматически незаметно преобразовываться в верхний регистр



Фильтрация ввода
  • Это поле для ввода возраста.
  • Вводить можно только цифры.
  • Если результирующая цифра меньше 0 или больше 150, она автоматически изменяется на ближайшую допустимую.
  • Здесь не учтена возможность вставки значения, например с помощью ctrl + v.
