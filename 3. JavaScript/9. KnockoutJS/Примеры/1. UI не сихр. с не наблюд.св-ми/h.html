Изменение НЕ наблюдаемых св-в модели не приводит к изменению их UI-подписчиков

  • Если менять значения в input'ах, св-ва модели тоже меняются
  • Но при этом модель не сообщает об этом своим подписчикам
  • Как следствие, значения UI-подписчиков на изменённые св-ва модели не меняются
  • Вывод: для не наблюдаемых свойств модели, синхронизация МОДЕЛЬ -> UI отсутствует
